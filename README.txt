# workplace_manager
ファイルはmasterブランチにあるのでmainから切り替えてください
使い方
git clone https://github.com/wada-nitwc/workplace_manager
でコピーする
cmdでworkplace_managerのディレクトリに移動する
python3で実行で必要なモジュールをインストールする
pip install -r requirements.txt
実行　python backend.py
実行してからhttp://127.0.0.1:5555/ にブラウザでアクセスする
python3 backend.py
